# Memorial Shrine

Create eternal digital memorials for loved ones on the blockchain.

## Features

- **Eternal Memorials**: Permanent on-chain tributes
- **Photo & Message Storage**: IPFS-backed media
- **Virtual Offerings**: Digital incense and flowers
- **Family Sharing**: Multi-user memorial access
- **Anniversary Reminders**: Automated notifications

## How It Works

1. Create a memorial shrine for a loved one
2. Add photos, messages, and life stories
3. Invite family members to contribute
4. Make virtual offerings to honor their memory
