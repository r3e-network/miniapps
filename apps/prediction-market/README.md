# Prediction Market 预测市场

Decentralized prediction markets on Neo N3.

## Overview

Create and participate in prediction markets for real-world events. Bet on outcomes and earn rewards for accurate predictions.

## Features

- **Create Markets**: Launch prediction markets for any event
- **Binary & Multi-outcome**: Yes/No or multiple choice markets
- **Automated Resolution**: Oracle-based outcome verification
- **Liquidity Pools**: AMM-style market making
- **Leaderboards**: Track top predictors

## How It Works

1. Browse active prediction markets
2. Buy shares in your predicted outcome
3. Wait for event resolution
4. Claim winnings if correct

## Technical Details

- **Category**: Other
- **Network**: Neo N3 Mainnet

## Development

cd apps/prediction-market
