# Charity Vault 慈善金库

Transparent charitable giving on Neo N3 blockchain.

## Overview

Charity Vault enables transparent, accountable charitable donations with full on-chain tracking. Donors can contribute to verified causes and track exactly how funds are distributed.

## Features

- **Transparent Donations**: All contributions recorded on-chain
- **Fund Tracking**: Real-time visibility into fund allocation
- **Verified Causes**: Curated list of legitimate charitable organizations
- **Tax Receipts**: Automatic generation of donation records
- **Multi-currency**: Support for GAS and NEP-17 tokens

## How It Works

1. Browse verified charitable causes
2. Select donation amount and currency
3. Confirm transaction via wallet
4. Track your contribution and its impact

## Technical Details

- **Category**: Finance
- **Network**: Neo N3 Mainnet
- **Permissions**: invoke:primary, read:blockchain
